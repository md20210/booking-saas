📋 How to Embed

Option 1: Simple Iframe

<iframe
  src="https://booking-saas-production-c352.up.railway.app/book/YOUR-EVENT-SLUG"
  width="100%"
  height="600"
  frameborder="0"
></iframe>

Option 2: JavaScript Widget

<script src="https://booking-saas-production-c352.up.railway.app/embed.js"></script>
<div id="booking-widget"></div>
<script>
  new MyBookingEmbed({
    containerId: 'booking-widget',
    apiUrl: 'https://booking-saas-production-c352.up.railway.app',
    apiKey: 'your-api-key',
    eventId: 'your-event-id'
  });
</script>
