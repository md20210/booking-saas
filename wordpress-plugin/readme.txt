== Description ==

**Booking SaaS Widget** ist das ultimative WordPress-Plugin für professionelle Terminbuchungen.

= Features =

* 🎨 **10 Design-Templates** - Von klassisch bis modern, minimalistisch bis verspielt
* 📊 **Google Ads Tracking** - Server-seitiges Conversion-Tracking mit GCLID
* 📅 **Calendar Integration** - Google Calendar & Microsoft Outlook/Office 365
* 🔗 **Teams & Meet** - Automatische Videokonferenz-Links
* 📱 **Fully Responsive** - Perfekt auf Desktop, Tablet & Mobile
* ⚡ **Blazing Fast** - Optimiert für Performance
* 🎯 **UTM Tracking** - Vollständige Marketing-Attribution
* 🛠️ **Easy Setup** - Shortcode & Gutenberg Block

Server-seitiges Conversion-Tracking für maximale Genauigkeit:
* GCLID (Google Click ID)
* GBRAID / WBRAID (iOS/Android)
* Enhanced Conversions mit First-Party-Data
* Automatische Attribution

= Verwendung =

**Shortcode:**
`[booking_widget event_slug="dein-event" template="modern-card"]`

**Parameter:**
* `event_slug` - Der Slug deines Event Types (erforderlich)
* `template` - Template Name (Standard: calendly)
* `width` - Breite des Widgets (Standard: 100%)
* `height` - Höhe des Widgets (Standard: 800px)

**Gutenberg Block:**
Nutze den "Booking Widget" Block im Gutenberg Editor für eine visuelle Konfiguration.

== Installation ==

1. Lade das Plugin hoch oder installiere es über das WordPress Plugin-Verzeichnis
2. Aktiviere das Plugin
3. Gehe zu "Booking SaaS" → "Settings"
4. Trage deine Booking SaaS API URL ein
5. Nutze den Shortcode Generator um dein Widget zu erstellen
6. Kopiere den Shortcode in deinen Beitrag oder deine Seite

== Frequently Asked Questions ==

= Benötige ich einen Booking SaaS Account? =

Ja, du benötigst einen Account bei Booking SaaS. Das Plugin verbindet sich mit deiner Booking SaaS Installation.

= Kann ich mehrere Widgets auf einer Seite einbetten? =

Ja, du kannst beliebig viele Widgets einbetten, auch mit unterschiedlichen Templates.

= Funktioniert das Plugin mit Caching? =

Ja, das Plugin ist vollständig kompatibel mit Caching-Plugins wie WP Rocket, W3 Total Cache, etc.

= Werden Tracking-Parameter automatisch weitergegeben? =

Ja, alle GCLID und UTM Parameter werden automatisch an das Widget weitergegeben.
